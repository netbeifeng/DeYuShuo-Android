# DeYuShuo_Android
一个德语学习类 安卓App 使用爬虫爬取网络资源，因此无后端 
Eine Android-App zum Deutschlernen verwendet Crawler, um Netzwerkressourcen zu crawlen, sodass es kein Backend gibt
